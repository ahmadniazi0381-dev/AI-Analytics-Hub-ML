About this project

This platform was built to bring together several machine learning and natural language processing techniques in one place. Instead of running separate scripts or Jupyter notebooks for each task, everything is accessible through a single web interface.

The goal is to make it easy to upload a dataset, choose a task, run the analysis, and see the results without needing to touch any code.

What this platform does

Problem it solves

Running machine learning experiments usually requires setting up Python environments, writing scripts, and managing output files manually. This platform removes that friction by wrapping each algorithm in a simple form — you just upload your data, set a few parameters, and click run.

What you can do

  • Find patterns in transaction data using Apriori
  • Train a neural network classifier on any tabular CSV
  • Ask questions over a block of text using transformer models
  • Generate text continuations from a prompt
  • Extract people, places, and organizations from text
  • Chat with an AI assistant for analysis help
